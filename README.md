# 												**GrassBot** 

#### **使用方法**

1、python必须>=3.11

2、创建虚拟环境   python -m venv venv

3、启动虚拟环境   venv\Scripts\activate   再运行  python run.py

4、获取你的userid

5、填写你的代理

#### **教程**

1、首先获取id控制台按一下f12
